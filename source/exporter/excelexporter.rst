.. _excel-exporter:

***********
Excel导出器
***********

.. _config:

配置参数
========

+--------+--------+--------------------------------------------------------+
| 参数名 | 类型   | 说明                                                   |
+========+========+========================================================+
| type   | string | excel                                                  |
+--------+--------+--------------------------------------------------------+
| file   | string | 必需，导出到此Excel文件                                |
+--------+--------+--------------------------------------------------------+
| sheet  | string | 必需，导出到此Excel工作表中（并不会覆盖已有Excel数据） |
+--------+--------+--------------------------------------------------------+

.. _usage:

使用说明
========

将数据导出到Excel表格中，不支持导出嵌套数据以及复合数据。
